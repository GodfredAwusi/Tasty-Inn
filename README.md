# Tasty Inn
This is a website for a restaurant take where customers can make orders of all kind and also choose whether to make online payment or pay on delivery.
### It must contain
* Sign up form
* Login form
* Order form
  * Dish order
  * Event order
* Homepage
* Recipe menu page
* Gallery
* About page
